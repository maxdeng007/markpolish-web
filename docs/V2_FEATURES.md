# MarkPolish Studio v2.0 - New Features

## 🎨 AI Image Generation (LIVE NOW!)

### What's New

Generate stunning images directly in MarkPolish Studio using AI! Powered by ModelScope's Z-Image-Turbo, you can now:

- **Generate images from text** in 5-15 seconds
- **Use English or Chinese** prompts
- **Choose from 4 size presets** (Square, Portrait, Landscape, Wide)
- **Download, save to library, or insert** directly into your content
- **100% FREE** with ModelScope API (rate limits apply)

### Quick Start

1. **Get API Key** (2 minutes):
   - Visit [ModelScope.cn](https://modelscope.cn)
   - Register/Login
   - Go to Profile → API Token
   - Generate and copy token

2. **Configure in App**:
   - Open MarkPolish Studio
   - Click **Images** tab
   - Click **AI Generate** sub-tab
   - Paste API key and save

3. **Generate Your First Image**:
   - Enter prompt: "A serene mountain lake at sunset, photorealistic"
   - Choose size: Square (1024×1024)
   - Click "Generate Image"
   - Wait 5-15 seconds
   - Download or insert!

### Example Prompts

**Product Photography**:
```
A sleek modern smartphone on a minimalist desk,
professional product photography, studio lighting
```

**Nature Scene**:
```
Misty forest at dawn with sunlight through trees,
atmospheric, peaceful, nature photography
```

**Portrait**:
```
Professional headshot of a confident businesswoman,
modern office, natural lighting, corporate photography
```

**Illustration**:
```
Cute cartoon cat on books, colorful illustration,
children's book art, warm colors
```

### Tips for Best Results

✅ **DO**:
- Be specific and descriptive
- Include style keywords (photorealistic, artistic, etc.)
- Mention lighting (natural, studio, golden hour)
- Specify quality (4K, high resolution)
- Use composition terms (close-up, wide angle)

❌ **DON'T**:
- Use vague prompts ("nice picture")
- Forget to mention style
- Request inappropriate content
- Exceed rate limits (50-100/hour)
