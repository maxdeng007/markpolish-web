# Quick Start: AI Image Generation

## 🚀 3-Minute Setup

### Step 1: Get API Key (2 minutes)
1. Visit **https://modelscope.cn**
2. Register/Login
3. Profile → **API Token**
4. **Generate** → **Copy** token

### Step 2: Configure (30 seconds)
1. Open MarkPolish Studio
2. Click **Images** tab (left sidebar)
3. Click **AI Generate** sub-tab
4. Paste API key → **Save**

### Step 3: Generate! (30 seconds)
1. Enter prompt: `"A serene mountain lake at sunset, photorealistic"`
2. Choose size: **Square**
3. Click **Generate Image**
4. Wait 5-15 seconds
5. **Download** / **Add to Library** / **Insert**

## 💡 Pro Tips

✅ **Be Specific**: More details = better results  
✅ **Add Style**: "photorealistic", "artistic", "cinematic"  
✅ **Mention Lighting**: "natural light", "golden hour", "studio"  
✅ **Include Quality**: "4K", "high resolution", "professional"  

❌ **Avoid Vague**: "nice picture", "something cool"  
❌ **Don't Rush**: Take time to craft good prompts

## 🎯 Size Guide

| Preset | Size | Best For |
|--------|------|----------|
| **Square** | 1024×1024 | Social media, general use |
| **Portrait** | 768×1024 | Vertical images, profiles |
| **Landscape** | 1024×768 | Headers, banners |
| **Wide** | 1536×768 | Ultra-wide banners |

---

## ⚡ Troubleshooting

**"API key not configured"**  
→ Add your ModelScope API key in AI Generate tab

**"API Error (401)"**  
→ Invalid API key, generate a new one

**"API Error (429)"**  
→ Rate limit hit, wait 5-10 minutes

**"Generation timeout"**  
→ Try again, check internet connection
